UI representation of the Conversation/Messages screen. Holds all the data required to show messages.

Parameters

isLoading

If we're loading (initial load).

isLoadingMore

If we're loading more data (pagination).

endOfMessages

If we're at the end of messages (to stop pagination).

messageItems

Message items to represent in the list.

selectedMessage

Currently selected message, to show the overlay.

currentUser

The data of the current user, required for various UI states.

newMessageState

The state that represents any new messages.

parentMessageId

The id of the parent message - if we're in a thread.

unreadCount

The count of messages we haven't read yet.
